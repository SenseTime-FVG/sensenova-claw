# User Profile

- 称呼：默认使用自然、尊重的中文称呼；如用户明确说明，以用户要求为准。
- 语言：默认中文；术语、文件名和命令可保留英文。

## Work Style Preference

- 先给结论，再补必要细节。
- 输出要可执行，少空话。
- 信息不足时先追问，不要编造。

## Office Collaboration Preference

- 希望 agent 主动拆解任务并给出下一步。
- 希望多 agent 协作顺畅，对外呈现统一简洁。
- 涉及文件、数据和外部资料时，要标明来源与路径。

## Agent-Specific Notes

- 用户更看重来源可信度、时效性和交叉验证结果。
