# 本地电脑自动化 AgentOS 产品需求文档（PRD）

---

# 一、产品定位与目标

本产品是一款运行在**本地电脑**上的自动化 AgentOS 系统，面向办公场景提供多源信息整合与自动化执行能力。

核心能力聚焦：

> 从多渠道输入 → 自动生成结构化大纲 → 逐页 HTML 内容生成 → 可视化交互修改 → 工程化转换为可编辑 PPTX 文件

系统本质是一个具备工具调用能力、沙箱执行能力与可扩展 Agent 架构的本地智能操作系统。

---

# 二、目标用户画像

## 2.1 核心用户

### 1）企业白领 / 咨询顾问

* 高频制作汇报 PPT
* 需要整合多来源材料
* 时间紧、结构能力要求高

### 2）产品经理 / 运营

* 从文档、数据、邮件生成方案汇报
* 需要快速输出结构化内容

### 3）学生 / 研究人员

* 从论文、网页资料生成汇报
* 需要结构重组能力

### 4）数据分析师 / 财务分析 / 商业分析

* 从 Excel/BI 报表/数据库导出数据生成经营分析或复盘汇报
* 需要自动产出指标解读、趋势/对比结论、异常点说明与可复用图表页
* 对数据口径一致性、可追溯性与结论可解释性要求高

输入：

* 项目状态汇报 PPT

> “帮我做一个关于 AgentOS 架构设计的 20 页 PPT”

* 生成结构
* 输出逐页内容
* 支持编辑

## 场景 5：本地文件夹批量生成培训课件

输入：

* 结构化培训 PPT

# 总结

本 AgentOS 产品核心竞争力在于：

* 本地安全运行
* 多源融合理解
* 结构化大纲生成能力
* HTML 中间表示层
* 工程化可编辑 PPTX 输出

产品的关键壁垒不在“生成内容”，而在：

> 如何构建一条稳定、可控、可编辑、可扩展的 PPT 工程化生产链路。

该系统本质是：

> 一个以 PPT 生产为核心应用场景的本地智能自动化操作系统。
